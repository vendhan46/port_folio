Tech Stack: Python, AES+Twofish, SHA-256, AWS, Google Cloud, Wasabi, OpenStack, Boto3, Google Cloud SDK, Matplotlib

The Multi-Cloud Secure Storage System is a robust and performance-driven project designed to securely store encrypted data across multiple cloud platforms including AWS, Google Cloud, Wasabi, and OpenStack. It employs a hybrid encryption model combining AES and Twofish algorithms to ensure strong data confidentiality, with keys derived using SHA-256. Encrypted files are distributed across various cloud providers to enhance redundancy and security. The system also features performance analysis by calculating upload/download speed and time, along with a comprehensive security evaluation based on encryption strength, authentication, access control, and compliance.
